# Word Harmony — Replit Setup

## How to run on Replit

1. Go to **replit.com** → Create account (free)
2. Click **"+ Create Repl"**
3. Choose **"Import from ZIP"** or **"Node.js"** template
4. Upload this ZIP file
5. Click the green **"Run"** button

Replit will automatically:
- Install all dependencies
- Build the React frontend
- Start the server

Your game will be live at your Replit URL (e.g. `https://word-harmony.yourusername.repl.co`)

## Share with friends
Copy your Replit URL and share it — anyone can open it in their browser and play!

## How to play
1. One player creates a game → gets a 5-letter session code
2. Share the code with 2+ friends
3. Everyone joins → host clicks Start
4. Submit words → get synonym cards → trade → buzz!
